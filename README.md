# Titanic Survival Prediction — Kaggle Challenge

A classification project predicting passenger survival on the Titanic, based on the classic Kaggle "Titanic: Machine Learning from Disaster" competition dataset.

## What it does

- Explores and visualizes the Titanic passenger dataset (`seaborn`) to understand survival patterns across features like class, sex, and age
- Trains a Decision Tree Classifier to predict survival
- Evaluates the model using accuracy score on a held-out test split

## Running it

Open `titanic.ipynb` in Jupyter and run all cells. Requires `pandas`, `seaborn`, and `scikit-learn`.

## Tech

Python, pandas, seaborn, scikit-learn
